## Exercise 3

Includes the R scripts for the exercise 3 tasks

- `1_initial_setup_ex3.R` includes all the initial set up of the exercise
- `2_splitting_downsampling_ex3.R` includes all the splitting, downsampling tasks for ex 3
- `3_yj_ex3.R` includes all the YJ manipulations and transformations in Ex 3
- `4_prediction_analysis.R` includes all the analysis of predicitons in Ex 3
- `results` contains datasets and outputs used in ex 2, including the splits, folds, training, testing and cleaned data
